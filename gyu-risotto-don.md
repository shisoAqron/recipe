# 牛リゾット丼

基本はリゾットを炊く手順で調理する。特殊調理素材は不要。

用意するもの
- 牛丼の素材: 牛皿やレトルトで十分。やるなら自作。汁は多いほうがよい
- コメ: 肉と相談。基本の炊き方では1合
- 油: リゾットを炊くのに使う。おすすめはオリーブ
- 白ワインか日本酒: リゾットを炊くのに使う。うまいほど良い。

手順
1. 牛皿やレトルトから牛煮込みと汁をわける
2. 汁に合わせたコメを用意する。基本は1つのセットに対して1杯分だと思われるので適宜汁を増やすか肉セットを増やす。タマネギなどの具は汁に入れておくと良さそう。
3. リゾットを炊く。手順は既知のため省略。
4. 炊きあがるタイミングに合わせて残しておいた肉を温める
5. コメを盛り付け、肉を盛る
