# 水菜の揚げのおひたし

## 材料(2人分)
- 水菜: 1/2束
- 油揚げ: 2枚
- 七味唐辛子や山椒: 適量
  - おすすめは[七味家](https://www.shichimiya.co.jp/SHOP/A0110.html)か[黒七味](https://shop.hararyoukaku.co.jp/i/040)
- 白だしのつゆ: 希釈して360ml分

## 手順

```mermaid
graph TD
A(水菜)
B(油揚げ)
C(白だしのつゆ)
D(七味唐辛子や山椒)

AA(洗った水菜)
A -- 砂を洗い落とす --> AA

AAA(切った水菜)
AA -- 5-6cmに切る --> AAA

BB(短冊にした油揚げ)
B -- 長さ3-4cm，幅7-8mmの短冊に切る --> BB

CC(煮立っただし)
C -- 炊く --> CC

CCC(油揚げを加えただし)
CC --> CCC
BB -- 油揚げを入れる --> CCC

CCCC(水菜を加えただし)
CCC -- 2-3分炊く --> CCCC
AAA -- 炊いた後入れる --> CCCC


CCCCC(水菜を少し煮たもの)
CCCC -- 水菜がしんなりするくらい煮る --> CCCCC

CCCCCC(盛り付けたもの)
CCCCC -- 盛り付ける --> CCCCCC

CCCCCCC(完成)
CCCCCC --> CCCCCCC
D -- 振る --> CCCCCCC
```

## メモ
- 汁多めにしておくと後で飲んだり麺類のおだしにできて便利
- 温かいままでも良いが，夏は冷やしてさっぱりさせるのもおすすめ
